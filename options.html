What This Setting Does

Return as Icon Link? If checked, the extension copies an HTML snippet like this: <a href="[crafted-url]" aria-label="Open in ChatGPT"> <img src="icons/open-in-chatgpt.svg" alt="" aria-hidden="true"> </a> If unchecked, the extension returns only the plain ChatGPT URL.
